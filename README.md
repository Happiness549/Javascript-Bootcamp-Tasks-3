# Signup Portal Project

## 📌 Overview
This is a simple **Signup Portal** built using HTML, CSS, and JavaScript.  
It includes form validation, asynchronous JavaScript concepts, and a student database simulation.

The project was built as part of a bootcamp assignment to practice:
- DOM manipulation
- Form validation
- Regular expressions (regex)
- Asynchronous JavaScript (`setTimeout`)
- Synchronous vs asynchronous execution

---

## ✨ Features

### 🧾 Form Validation
- Username cannot be empty
- Email must be valid format
- Password must be at least 8 characters
- Terms checkbox must be ticked
- Error messages display under each input

### ⏳ Asynchronous Feature
- Student records are stored in a fake database (`studentDatabase`)
- Data is displayed after a **3-second delay** using `setTimeout`

### 🔁 Synchronous Demo
- Demonstrates JavaScript execution order using console logs
